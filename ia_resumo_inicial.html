O hardware é todo componente físico (tocavél) do computador, ou seja, hardware inclui peças como placa mãe; placa de video; processador; entre outros aparatos eletrônicos que fazem o equipamento funcionar. o software já é o completo oposto, é a parte referente ao sistema, ou seja, programas e aplicativos. Rede de computadores é uma troca de dados entres os dispositivos com intuito de um melhor gerenciamento de dados. Esse compartilhamento de dados pode acontecer com ou sem fio. Sistema operacional é a parte que executa as tarefas, armazena, cria e cópia.O sistema operacional que permite iniciação do seu dispositivo e faz o o contato do usúario com o softawere, com o hardware e com a rede.
