Qual a diferença entre algoritmo e programa?

Um algoritmo nada mais é do que comandos em uma sequência lógica que têm como objetivo programar a máquina em uma linguagem natural para resolver problemas do cotidiano. O programa é um conjunto de algoritmo organizados em uma linguagem binária, ou seja, uma linguagem de zeros e uns. Essa linguagem, é a única que o computador consegue ler, por isso, existe essa necessidade de transformar todo algoritmo em uma linguagem binária. Um algoritmo pode ser escrito em várias linguagens diferentes, como por exemplo, HTML e PYNTHON, e independente de em quantas linguagens diferentes o algoritmo esteja, tudo será traduzido para a linguagem binária, contanto que tenham lógica, tudo funcionará perfeitamente bem.
