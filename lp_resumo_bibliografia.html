Algoritmo

Algoritmo nada mais é do que comandos em uma sequência lógica que visam a resolução de algum problema. Algoritmo e lógica de programação estão interligados, de forma que, cada algoritmo deve ser colocado de forma lógica para a resolução, caso contrário, o processo de modificação para o código binário apresentará falhas e inconsistências. Um exemplo bem simples e real para a maioria dos brasileiros, sem dúvidas, é uma receita qualquer que, com símbolos e de uma forma lógica e sequencial, determina os passos para a realização de um prato em especifico.
